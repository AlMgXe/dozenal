Documentation

Each of the dozenal programs has an extensive man page; some also have detailed other documentation. Check below for the specifics.

  • dec and doz These programs are mostly self-explanatory, and what isn't obvious the manual should be able to cover. Still, they operate so similarly that an additional tutorial was written.
    • doz manual
    • dec manual
    • dec/doz Tutorial
  • dozdc A fully-functional RPN calculator. The manual is complete, but the tutorial will be welcome, especially if one isn't familiar with postfix notation.
    • dozdc manual
    • dozdc Tutorial
  • tgmconv A metric system converter, that works in U.S. customary, imperial, SI metric, and the dozenal TGM system.
    • tgmconv manual
    • tgmconv Tutorial
